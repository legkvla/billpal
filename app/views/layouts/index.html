Про проект

Приняв решение участвовать в AngelHack Moscow, мы согласились испытать свои силы, начав новый проект с нуля. Выбирая идею, мы решили, что реализовывать проект с надуманной идеей будет не так интересно, как тот, что решает реальную проблему. Удобный рассчет с независимыми подрядчиками прийдется актуалным многим нашим коллегам, и для нас это отличная мотивация.
Весь проект написан на Ruby on Rails, на фронтенде используется AngularJS. Ввод средств осуществляется через api paysio.com, выплаты производятся через JustGateway. В верстке использован Twitter Bootstrap c коммерческими шаблонами. Вся логика сервиса написана целиком в течении хакатона.
Важно: в течении конкурса все операции производятся с виртуальной валютой изпользуя только тестовые доступы гейтвеев. Получение продакшен доступа связано с бюрократическими вопросами и не может быть решено в течении выходных.
